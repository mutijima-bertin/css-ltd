# Creative Sound Studio Ltd

**Multimedia Production House — Kigali, Rwanda**

Creative Sound Studio Ltd is a professional multimedia and audio-visual company based in Kigali, Rwanda. Founded by **Nkurunziza Jabo**, the studio operates as an independent multi-platform production house dedicated to discovering, recording, and promoting unrepresented local and regional African talent.

---

## About the Founder

**Nkurunziza Jabo** is a Rwandan multimedia creator and the foundational artist behind Creative Sound Studio Ltd. He operates as a digital creator, photographer, and audio-visual professional, leveraging the studio as his main operational imprint to offer professional digital media services across Rwanda. His creative work encompasses high-quality event photography, digital photo editing, and technical multimedia production.

## Core Focus Areas

### 🎵 Talent Discovery
A creative launchpad designed to discover, record, and promote unrepresented local and regional African talent.

## Problems We Solve

| Problem | Solution |
|---------|----------|
| **Portfolio Chaos** — Scattered media across social media and phone storage | Centralized high-fidelity digital showroom with crisp photography grids, embedded video, and audio streams |
| **Booking Inefficiency** — Lost leads from slow WhatsApp responses | Automated 24/7 booking calendar with real-time studio availability |
| **No-Shows & Payment Tracking** — Clients book but don't show; manual MoMo tracking | Integrated MTN MoMo & Airtel Money payments via Flutterwave with upfront deposit requirement |
| **Limited Talent Reach** — Word-of-mouth only for demo submissions | Structured Talent Submission Portal for vocalists and instrumentalists |
| **Poor Search Visibility** — No Google presence for local multimedia services | Local SEO-optimized, mobile-friendly website |

## Getting Started

### Prerequisites
- Node.js >= 18
- npm, yarn, pnpm, or bun
- MariaDB

### Development

```bash
# Install dependencies
npm install

# Run the development server
npm run dev

# Build for production
npm run build

# Start production server
npm run start

# Lint
npm run lint
```

Open [http://localhost:3000](http://localhost:3000) to view the application.
